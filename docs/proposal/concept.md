## 1. プロジェクトの目的

### 現状
既存サイト（https://www.yumehoken.jp/）は WordPress（Lightning Theme）で構築されており、保険商品案内・お知らせ・社会貢献を中心に情報発信しているものの、

- **終活ステーション**（独自サブブランド・木庭理恵氏担当）の存在がほぼ伝わっていない
- **法人向けサービス**（リスクヘッジ・経営コンサル）のページが**存在しない**
- **採用情報ページが存在しない**ため、人材獲得の機会損失
- **ログハウスオフィス**という最大の差別化要素が訴求できていない
- メンバー紹介・代表メッセージが**画像化**されており、SEO・アクセシビリティに不利
- LINE 公式アカウント @942hvyou・ISO 9001:2015 認証など、**既存資産が表に出ていない**

### 目的
**「保険を売る会社」から「夢と安心をつなぐ生涯パートナー」へ**

ゆめほけんの本質である「ゆめサポート」（プロフェッショナル連携）と「あんしんつなぎ」（保険総合コンサル）を、訪問者の属性別に最適化された **4つの入口** で提示する **ポータルサイト** に刷新します。

### コア・コンセプト
> **「ログハウスから届く、夢のおまもり。」**

ゆめほけんの事務所は北九州市小倉南区にある **ログハウス**。木のぬくもりと自然光に包まれたこの空間は、保険代理店としての硬さを和らげ、**「相談しに行きたくなる場所」** という他社にない体験を提供しています。

## 4. デザインコンセプト

### トーン＆マナー
- **保険代理店としての誠実さ × ログハウスの温かみ × 自然のナチュラル感**
- 「**かたい**」と「**やわらかい**」の中間（ITサービスほどシャープではなく、町の保険代理店ほど古くない）

### カラーパレット
ロゴから抽出した自然由来のパレット：

| 役割 | カラー | サンプル |
|---|---|---|
| プライマリ（葉のフレッシュグリーン） | `#95C11F` | ロゴの葉そのもの |
| セカンダリ（木のウッドブラウン） | `#8B5A2B` | ロゴの幹そのもの |
| アクセント（キャラメル茶） | `#C8956B` | ロゴの「Dream Insurance Inc.」字色 |
| 法人ハブ用（ログハウスのグレーブルー） | `#6B7B8C` | ログハウス外観の壁色 |
| ベース | `#FAF7F0` クリーム / 白 | 木の温かみと清潔感 |

### タイポグラフィ
- **見出し**: Noto Serif JP（明朝体）— 信頼感と木のぬくもり
- **本文**: Noto Sans JP（ゴシック）— 高齢層も含めた可読性
- **欧文**: Inter / Outfit — ロゴと相性の良いモダンサンセリフ

### ビジュアル方針
- **主役の写真**: ログハウス外観・薪ストーブ・木の内装・メンバーの笑顔
- **イラスト**: 必要に応じてAI生成（水彩・温かみ・ナチュラル）
- **アイコン**: Lucide React の線画系アイコンを基本に、葉モチーフを差し色で

## 7. 既存サイトとの主な差分

| 領域 | 既存サイト | 新サイト |
|---|---|---|
| 入口 | フラットな6メニュー | **4タイプ別ポータル** |
| 終活ステーション | 「お客様の声」内で言及のみ | **独立サブブランドサイト** |
| 法人向け | なし | **専用ハブ新設** |
| 採用 | なし | **専用ハブ新設** |
| 代表/メンバー紹介 | 画像のみ | **テキスト化＋顔写真**（SEO/a11y強化） |
| LINE | サイトに記載なし | **全ページCTAで配置** |
| ログハウス | 訴求弱い | **ヒーロー級の差別化要素として大きく** |
| URL構造 | `/about-2`, `/lettaers` 等の汚れ | **クリーンな再設計** |
| ニュースレター | バックナンバー一覧のみ | **記事単位でWEB読了可能**（SEO効果） |
| CSR | 簡素な紹介 | **5パートナーごとにストーリーリッチ化** |
| ISO認証 | サイトに非表示 | **信頼バッジとして表示** |
| FD宣言の数字 | 別ページに埋もれている | **TOPで信頼バッジとして強調** |

## 8. 移行戦略（既存資産の保全）

### URL リダイレクト計画（旧 → 新）
```
/about-2          → /about
/product-2        → /personal
/news-2           → /news
/newslettaers-list → /newsletter
/jouhou_securty   → /policies/security
/kojin_kihonhushin → /policies/privacy
/fddeclaration2024 → /policies/fd
/csr/*            → /csr/*（パスはそのまま、内容リッチ化）
/accidentprocedureguide, /night_holiday, /towing_service → /support
/moushikomi       → /apply
```
→ `next.config.mjs` の `redirects()` で実装

### SEO 対策
- 既存サイトの上位流入キーワードを把握（GSC）
- → **GSC（Google Search Console）はユーザー本人が設定**する
- 構造化データ（LocalBusiness, FinancialService）を実装
- メタタグ・OG画像を全ページで最適化
